This is an Alpha testing version of the latest features on Course Picker.

This Alpha testing version contains the following new features (with more features to come in the near future!):

Export your Picker selections to Google Calendar.
Choose from Fall, Spring, and IAP terms.
How many Course Picker users are enrolled in a class.
Browse all classes at once.
Collapse/uncollapse course descriptions for easier viewing.

Course Picker is a scheduling tool to help MIT students plan their schedules using a semantic data source and a clean user interface. Making subject selections with Picker might actually save some brain cells for your coursework!

However, please note that this system is not part of WebSIS and the Institute's scheduling system. Using this system does not register classes on your behalf!

As an Alpha release, we ask for your help in improving this tool, please email us with any suggestions or comments.
